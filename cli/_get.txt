Get a memo.

Usage: docbase get <DOMAIN> <MEMO> [OPTIONS]

Arguments:
  DOMAIN                Domain name of the team
  MEMO                  Memo ID or URL to get

Options:
  --json                Output as JSON    
  -h, --help            Show help

Examples:
  docbase get my-domain 1234

  docbase get my-domain https://{domain}.docbase.io/posts/1234
